Estrutura de Repetição

While -- enquanto o teste for verdadeiro vai fazer o bloco de repeticao
Until -- enquanto for falso nao vai fazer nada

```ruby
i = 1

while i <= 50
    puts "bruno - #{i}"
    i += 1
end
```

```ruby
#aqui o 1 esta com o valor de 50 
# i == 50

until i <= 1
    puts "sou falso - #{i}"
    i -= 1
end
```
